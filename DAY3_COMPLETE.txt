================================================================================
                        Q_INTENTCOMPILER ALGORITHM
================================================================================

Core Innovation: Minimax game theory applied to trading

INPUT: Market signals from VISION + patterns from GEMINI
   └─ foresight: {affinity, drift_risk}
   └─ deltas: {momentum, volume, sentiment, ...}
   └─ timestamp: ISO 8601

PROCESS:

1. BUILD FRACTAL TREE
   ├─ Create node for each market trait
   ├─ Assign value (0-1) to each node
   ├─ Set depth based on risk (drift_risk * 10)
   └─ Result: Tree structure representing market state

2. APPLY MINIMAX OPTIMIZATION
   ├─ Model as 2-player game:
   │  ├─ MAX (KLOUTBOT): Maximize profit
   │  └─ MIN (Market): Minimize our profit
   ├─ For each node, evaluate alternative moves
   ├─ Select optimal strategy considering worst case
   └─ Result: Conservative but optimal decision

3. RECURSIVE IMPROVEMENT (3 iterations)
   ├─ Generate alternative tree configurations
   ├─ Select highest-utility alternative
   ├─ Apply efficiency optimizations
   ├─ Repeat N times for refinement
   └─ Result: Self-improving strategy

4. GENERATE ACTION
   ├─ Determine optimal action: BUY, SELL, HOLD
   ├─ Calculate position size from confidence
   ├─ Select asset pair from dominant trait
   └─ Result: Actionable trading parameters

OUTPUT: DecisionTree with complete reasoning trace
